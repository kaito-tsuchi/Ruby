# Ruby

「ゼロから分かるRuby超入門　著:五十嵐邦明、松岡浩平」の参考書を元に、Rubyの勉強を行います。

以下のようなスケジュールで、Rubyの基礎を学んでいきます。

2/1：環境構築

windowsにRubyをインストールし、VSコードでコードを書き、コマンドプロンプトで実行するように環境を整えました。また、VSコードをGitHubに連携しました

2/2～2/3：Rubyの基本的な知識

まず、Rubyの基礎知識として、数値や文字列の表示方法や四則演算、オブジェクト、変数について簡単なコードを書きつつ学びました。この単元は、今まで学んだCやJavaなどの知識を活かすことができたので、比較的簡単でした。

2/4：条件分岐

ifやunless、else節、case、timesメソッド、whileの書き方や機能をコードを書きつつ学びました。この単元も、今まで学んだ言語の知識を活かすことができたので、スムーズに学習することができました。しかし、timesメソッドについては、初めて学んだ内容でした。このメソッドは、決まった回数だけ処理を繰り返すことができるので、とても便利で使い道も多そうだと思いました。

2/5：配列

配列について、要素の取得や追加、削除から配列同士の演算、繰り返し処理の方法を学びました。特に印象的なのは、eachメソッドを用いた繰り返し処理です。途中で終わらせるbrekや繰り返しの次の回に進むnext、範囲を指定する（）等、初めて学ぶことが多くありました。ここの部分はよく復習しておきたいです。

2/6：メソッドの利用

ここの単元では、Rubyに用意されている様々なメソッドを調べ方を学びました。同じ名前のメソッドでも末尾に！がつくと機能が変化することがあるのがとても印象的でした。この単元で用いたのは、配列の要素数を得る -sizeメソッド、配列の全要素の合計を得る -sumメソッド、配列から重複した要素を取り除く -uniqメソッド、ランダムに要素を取得する　－sampleメソッド、配列の要素を順に並び変える -sortメソッド、配列の要素を逆順にする -reverseメソッド、配列中の文字列を連結する -joinメソッド、文字列を分割して配列にする -splitメソッド、配列の各要素を変換した配列を作る -mapメソッド等である。

2/7：ハッシュ

この単元では、ハッシュという複数のオブジェクトをまとめて入れることができる道具について学びました。ハッシュは「キー」と「値」のセットで、複数のデータを整理し扱っており、データを効率よく操作することができるのが特徴です。主に学んだことは、このハッシュの定義方法や名前の付け方、ハッシュから値を取得する方法、キーと組みの追加、削除方法、ハッシュの要素の繰り返し処理の方法などです。
